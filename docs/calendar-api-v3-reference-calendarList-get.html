On this pageRequestHTTP requestParametersAuthorizationRequest bodyResponseExamplesTry it Note: Requires authorization. Returns a calendar from the user's calendar list. Try it now or see an example. Request HTTP request GET https://www.googleapis.com/calendar/v3/users/me/calendarList/calendarId Parameters Parameter name Value Description Path parameters calendarId string Calendar identifier. To retrieve calendar IDs call the calendarList.list method. If you want to access the primary calendar of the currently logged in user, use the "primary" keyword. Authorization This request requires authorization with at least one of the following scopes: Scope https://www.googleapis.com/auth/calendar.readonly https://www.googleapis.com/auth/calendar For more information, see the authentication and authorization page. Request body Do not supply a request body with this method. Response If successful, this method returns a CalendarList resource in the response body.
